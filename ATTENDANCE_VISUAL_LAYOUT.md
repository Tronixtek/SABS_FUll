## ğŸ“Š Data Display Formats

### Date Format
- **Display**: `MMM dd, yyyy` (e.g., "Jan 15, 2024")
- **Input**: `yyyy-MM-dd` (e.g., "2024-01-15")

### Time Format
- **Display**: `hh:mm a` (e.g., "09:30 AM")
- **Fallback**: "N/A" if time is missing

### Number Format
- **Work Hours**: `X.XX hrs` (e.g., "8.50 hrs")
- **Overtime**: `X.XX hrs` (e.g., "0.50 hrs")
- **Late Minutes**: `X min` (e.g., "15 min")

### Status Format
- **Display**: Badge with colored background
- **Text**: Capitalized (e.g., "Present", "Absent")
